SC제일은행

SC제일은행 웹사이트는 HTML,CSS,Javscript,JQuery 에서 배운 기초를 토대로 클론하기 좋은 웹사이트라고 판단되어 선정하였습니다. 입문자로서 너무 어려운 웹사이트를 바로 시작하기 보다는 SC제일은행과 같은 웹사이트로 시작하여 구조를 파악하고 간단한 슬라이드 및 네비게이션 등의 효과를 직접 코드를 작성하고 적용해보는데 초점을 두었습니다.

언어 및 프로그램

무엇을 배웠나요?

sc제일은행의 스타일 및 디자인 구조를 분석하고 클론함으로써, 효과적인 CSS 스타일링과 웹 페이지 레이아웃에 대한 이해를 강화할 수 있었으며 사용자 경험을 향상시키는데 필요한 효과적인 네비게이션과 사용자 인터페이스 디자인을 배울 수 있습니다. 또한, 사이트의 동적 요소를 구현하기 위해 자바스크립트를 사용하면서, 이벤트 처리, 애니메이션, 그리고 사용자와의 상호작용에 대한 기본 개념을 익힐 수 있었습니다.

HTML

# HTML을 사용하여 웹 페이지의 구조를 정의하고 콘텐츠를 표시할 수 있습니다

CSS

# CSS(Cascading Style Sheets)는 HTML 문서의 스타일과 레이아웃을 디자인하기 위한 스타일 시트 언어입니다. 다양한 디자인 요소를 조절하여 웹 페이지를 꾸밀 수 있습니다

Slider Effect

# Slick Slider는 웹 페이지에서 간편하게 사용할 수 있는 반응형이며 다양한 설정 옵션을 제공하는 이미지 및 콘텐츠 슬라이더 라이브러리로 Slick slider를 활용하여 다양한 slider를 구현할 수 있습니다.

Hover Effect

  • 고객우대혜택
  • 금융사기예방
  • 환율
  • 금리
  • 이용시간
  • 수수료

# hover를 사용하여 다양한 스타일 변경을 할 수 있으며, 이를 활용하면 웹 페이지의 사용자 경험을 더욱 향상시킬 수 있습니다.

navigation 활용

# main,sub nabigation bar를 활용하여 웹 페이지의 주요 섹션이나 페이지로의 빠른 액세스를 가능하게 합니다. 이를 통해 사용자는 웹 사이트를 효과적으로 탐색할 수 있고, 핵심 콘텐츠로 쉽게 이동할 수 있습니다.
