##  프로젝트 소개


본 프로젝트는 개인 웹사이트를 제작하여 포트폴리오를 만든 것입니다.
해당 웹사이트에서는 작성자의 디자인 기술 및 능력을 확인 할 수 있는 디자인들이 녹아져 있습니다.

## 주요 기능


UI/UX 향상:

- 레이아웃 및 디자인 개선
- 사용자 경험을 향상시키는 애니메이션 또는 효과 추가
- 반응형 디자인 지원

## 사용 언어
  - HTML
  - CSS
  - Jquery
  - Javascript
